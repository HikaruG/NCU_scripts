To make it runnable, one needs to have: 

cellphone.py 
cellphone_phonecallsys.py
cellphone_contact.py
cellphone_calendar.py
country_code.xls

these 5 files placed in the same folder;


The cellphone class has several sub-classes: 
1. the phonecall system 
	for now, it can only call from dialing
2. the contacts system
	it can create a new contact, display the current contact list, and delete a contact
3. the calendar system
	it can create a new event, present it and delete the calendar
